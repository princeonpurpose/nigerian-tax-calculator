A modern SaaS web application for calculating Nigerian taxes under the 2025/2026 tax reform framework.

## Features
- Personal Income Tax (PIT) calculator
- Corporate Income Tax (CIT)
- VAT & CGT modules
- User authentication (Email & Google)
- Saved calculation history
- Dashboard with charts
- Filing reminders with email notifications

## Tech Stack
- Vite + React
- Supabase (Auth + Database)
- Netlify (Hosting)
- Chart.js
- Node / Edge Functions

## Setup

1. Clone repo
2. Install dependencies  
   `npm install`

3. Create `.env` file using `.env.example`

4. Run locally  
   `npm run dev`

## Database
Run `schema.sql` inside Supabase SQL Editor.

## Deployment
Hosted on Netlify.  
Set environment variables in Netlify dashboard.
